# Modulo Paqueteria - Arquitectura de Sistemas II

## Tecnologias
- **Node.js** (API)
- **Express** (Framework)
- **MySQL** (Base de datos)
- **Sequelize** (ORM)
- **Nodemon** (Desarrollo)

## Instalacion

1. Clonar el repositorio:

```bash
git clone https://github.com/JulioTzicap/modulo-paqueteria.git
cd modulo-paqueteria
```

2. Instalar dependencias:

```bash
npm install
```
